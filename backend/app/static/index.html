Source Grounding Auditor

Citation-only mode：只分析带 citation 的句子或段落。未带 citation 的内容不会进入 claims 或 ratios。

默认抓取显式 URL，并会搜索缺失 URL 的来源说明。仍可在 API 请求里传 provided_sources 做精确测试。

比例摘要

Problematic citations

这些是基于 cited claims 判断出的、作者或模型实际主张的、重要的、且证据关系存在实质问题的 citation。

Attribution-supported citations

这些 cited claims 的 source 支持“某来源说过这件事”，但不等于被转述内容本身已经被一手事实证明。

Audit-limited citations

这些 citations 不是 problematic，只是本轮没有足够 source body 或网页抓取来完成 source support check。

已排除的上下文、示例和限制说明

这些内容不是作者实际主张，或者只是 caveat、source pointer、用户问题、章节标题、反例。
